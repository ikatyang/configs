# iTerm2

## theme

- install https://github.com/sindresorhus/iterm2-snazzy
- Preference -> Profiles -> Color -> Color Presets... -> Snazzy

## Preference

- General
  - tmux Integration
    - [Open tmux windows as tabs in existing window]
- Appearance
  - Tabs
    - [x] Stretch tabs to fill bar
  - Dimming
    - [ ] Dim inactive split panes
